Dependencies per module

As of version 1.0.12, logback requires JDK 1.6 to build, however, it is intended to run under JDK 1.5 (with the exception of SSL related functionality).

Each logback module has a different set of dependencies. These are listed below in a separate table per module.

SMTPAppenderBase and derived classes
  • JavaMail API version ${javax.mail.version},
  • JavaBeans Activation Framework (JAF) version 1.1
